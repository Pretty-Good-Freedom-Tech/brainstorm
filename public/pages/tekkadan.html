Tekkadan Social Graph Analysis

Tekkadan Social Graph Analysis looks at interactions between your social graph and another user's social graph.

Each Graph is the result of a cypher query that takes two inputs: owner pubkey (you) and this user's pubkey.

Some queries, such as Frens, Fans, and Idols, use only the user pubkey, while others use both pubkeys. Those are still considered TSG because the results of those queries are used in more complex TSG lists.

Each Graph has its own unique cypher query.

Each Query returns a list of pubkeys.

The results of the query are displayed in table format. (Techically it's a list, not a graph?)

There are 5 TSG Lists:

  • Frens
  • Fans
  • Idols
  • Recommended Follows (You → Owner)
  • Recommended Follows (Owner → You)
